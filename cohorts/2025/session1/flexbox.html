üöÄ What is Flexbox?

Flexbox (Flexible Box Layout) is a modern CSS layout method that makes it incredibly easy to arrange items in a container. Think of it as a smart way to organize elements that automatically handles spacing, alignment, and responsiveness.

üí° Key Concept: Flexbox works with two main components:
  • Flex Container (Parent): The element with display: flex
  • Flex Items (Children): The direct children of the flex container

üìö Quick Reference & Cheat Sheet

Container Properties (Parent)

  • display: flex - Enable flexbox
  • flex-direction - Main axis direction
  • justify-content - Main axis alignment
  • align-items - Cross axis alignment
  • flex-wrap - Wrapping behavior
  • gap - Space between items

Item Properties (Children)

  • flex-grow - How much to grow
  • flex-shrink - How much to shrink
  • flex-basis - Initial size
  • flex - Shorthand for above 3
  • align-self - Override container alignment

üéØ Most Common Patterns:

  • Center anything: display: flex; justify-content: center; align-items: center;
  • Space between items: justify-content: space-between;
  • Equal width columns: flex: 1; on each item
  • Responsive cards: flex: 1 1 300px; with flex-wrap: wrap;
  • Sidebar layout: flex: 0 0 200px; for sidebar, flex: 1; for main

üö® Common Mistakes to Avoid:

  • Forgetting that align-items works on the cross axis, not main axis
  • Using float or display: inline-block on flex items
  • Not understanding that flex-direction: column switches the axes
  • Overusing !important instead of understanding flex properties
